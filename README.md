10 Class in dataset:
* Bacterial Spot
* Early Blight
* Healthy
* Late Blight
* Last Mold
* Mosaic Virus
* Septoria Leaf Spot
* Spider mites Two-spotted Spider Mite
* Target Spot
* Yellow Leaf Curl Virus
